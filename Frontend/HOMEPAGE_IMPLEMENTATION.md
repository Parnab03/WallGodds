## 🚀 **How to Run**

1. **Install dependencies:**
   ```bash

   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **View in browser:**
   - Open `http://localhost:5173`
   - The homepage will display exactly as shown in the screenshot
